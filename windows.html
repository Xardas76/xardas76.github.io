Windows (с англ. — «Окна») — семейство коммерческих операционных систем (OC) корпорации Microsoft, ориентированных на управление с помощью графического интерфейса. Изначально Windows была всего лишь графической программой-надстройкой для распространённой в 1980-х и 1990-х годах операционной системы MS-DOS.

Согласно данным ресурса Net Applications, по состоянию на август 2014 года под управлением операционных систем семейства Windows работает около 88 % персональных компьютеров.

История и разновидности

Даты выхода ОС Windows для персональных компьютеров (ПК)

  • Windows 1.0 (1985)
  • Windows NT 3.1 - 3.51 (1993-1995)
  • Windows 95 — Windows 4.0 (1995)
  • Windows 98 — Windows 4.1 (1998)
  • Windows 2000 — Windows NT 5.0 (1999)
  • Windows XP — Windows NT 5.1 - 5.2 (2001-2005)
  • Windows Vista — Windows NT 6.0 (2006)
  • Windows 7 — Windows NT 6.1 (2009)
  • Windows 8 — Windows NT 6.2 (2012)
  • Windows 10 — Windows NT 10.0 (2015-2019)

Программы

Вместе с Windows на компьютер устанавливается набор программ, необходимых для повседневного использования:

  • Калькулятор
  • Редактор Notepad (блокнот) для работы с текстовыми файлами
  • Редактор WordPad для работы с документами
  • Редактор картинок Paint
  • Браузер Edge или Internet Explorer для работы в Интернет
  • Проигрыватель видео и музыки
Можно устанавливать другие программы и игры из Интернета или с дисков. Такие программы также могут быть платные.

В последних версиях Windows есть сборки, отличающиеся функционалом и ценой: Home Edition — сборка для домашнего использования с простой конфигурацией. Professional — сборка с дополнительным функционалом для более продвинутых пользователей. Enterprise — корпоративная версия для организаций.

Как получить?

Операционная система Windows платная. Если вы покупаете компьютер с уже установленной Windows, то часть денег вы платите за операционную систему. Для подтверждения того, что вы являетесь владельцем Windows, может потребоваться лицензионный ключ. Такое бывает, например, если вы переустановили Windows. Лицензионный ключ это набор символов вида "XXXXX-XXXXX-YYYYY-YYYYY-ZZZZZ". Он может быть указан в наклейке на корпусе вашего компьютера или на диске с Windows.
